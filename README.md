# Healthy-Meal-Planner
AI Healthy Meal Planner is a web application that generates personalized, nutritious meal plans based on user dietary preferences, restrictions, and calorie goals. It aims to simplify healthy eating with AI-driven recipe suggestions and shopping lists.
HealthyMealPlanner Mini-Site
#Overview
HealthyMealPlanner is a responsive web mini-site designed to help users plan nutritious meals, track nutrition, and explore healthy recipes. This project is a simple, clean, and accessible frontend-only website built using HTML and CSS, featuring user-friendly forms, accordion FAQs, and tabbed contentâ€”all created without JavaScript.

#Features
Clear navigation with anchored sections: About, Features, Join Us, FAQ, Plans, Contact

Customizable signup form with interests, subscription plan selection, and updates toggle

Accordion style FAQ for quick information access

Responsive tabbed interface for subscription plan details

Mobile-friendly design with accessible form elements and semantic HTML

#Usage
Open index.html in any modern browser to view the site

Styles are included in style.css; link it in the <head> of the HTML file

No JavaScript required; purely HTML and CSS based

Forms are non-functional placeholders (action="#"), for demonstration

#Technologies
HTML5 semantic elements for better structure and accessibility

CSS3 with Flexbox and media queries for responsive design

Accessible form controls and interactive UI components built with CSS only

#How to Customize
Update textual content directly in HTML sections to fit your project

Modify or extend CSS in style.css to change colors, layout, or add new components

Connect the forms to backend services or JavaScript for real-world data capture as needed

#Ideal For
Beginners learning responsive design and CSS-driven UI components

Demonstrations or prototypes of meal planning or wellness websites

Frontend developers focusing on accessible, simple, and responsive markup
